About this NDoc in 2014

Since the early 2000ers, NDoc is a well known program to produce HTML, CHM, or PDF files from XML documentation in .NET style and the corresponding assemblies. However, development on this project stopped in 2003 (refer to http://www.sourceforge.net/projects/ndoc).

This distribution is a recompilation of the project with VS 10 and .NET 4.0. The documenters MSDN (for HTML and CHM output) and LATEX (for PDF or DVI output) should work properly.

About NDoc in 2003

NDoc generates class libraries documentation from .NET assemblies and the XML documentation files generated by the C# compiler (or an add-on tool for VB.NET such as VBCommenter).

NDoc uses add-on documenters to generate documentation in several different formats, including the MSDN-style HTML Help format (.chm), the Visual Studio .NET Help format (HTML Help 2), and MSDN-online style web pages.
