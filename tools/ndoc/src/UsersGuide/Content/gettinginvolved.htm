Contribute to the Success of NDoc

There are many ways you can become involved in the NDoc development effort.

  • Participating in the ndoc-users email list is the easiest way to share your tips and tricks for using NDoc with other users.
  • If you find bugs or have suggestions for new feature please use the NDoc tracker database so that we can keep current with our user's needs.
  • If you have some time to spare, become part of the NDoc developer team. There are plenty of new features we are working on implementing, and there are always bugs to fix, and the more people we have working on NDoc, the better it becomes. To join the effort, contact one of the project admins via the NDoc SourceForge foundry, and we will get you set up.
  • If you've implemented your own custom tags you can share your work with other NDoc users via the NDocContrib collection.

Thanks for using NDoc.
